Rate Limiting Permits
=====================

We have a general-purpose rate limiting system.

```
CREATE TABLE Permit (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  type VARCHAR(255) NOT NULL,
  value VARCHAR(255) NOT NULL,
  acquire_at TIMESTAMPTZ NOT NULL
);

CREATE UNIQUE INDEX PermitTypeValueAcquireAt ON Permit (
  type,
  value,
  acquire_at
);
```

Each time a customer attempts a rate-limited action (like linking an email address), we track that.
