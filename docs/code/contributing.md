Contributing to Wasmo
=====================

Wasmo OS (this repo)
--------------------

We don't use large language models to author the code, documents, graphics, or other assets in this
repo.

We feel that conventional software engineering is the best way to build software that’s small and
simple.


Wasmo Applications
------------------

We’re not opinionated on how Wasmo applications are developed!

We want you to build with the tools that you prefer.

Wasmo applications run in a secure sandbox, so it's a great place to experiment with technology and
have fun.


A Note on Copyright
-------------------

We use copyright to license our work on Wasmo. It is our understanding that generated code is
[not subject to copyright in Canada][copyright].
