Milestone 2
===========

Proposed on 2026-03-20 by Jesse
✅ Status: complete on 2026-04-28

Journal App
-----------

Build out the journal web app with the following capabilities:

 - Display a list of entries
 - Create or edit an entry
 - Attach an image to an entry
 - Publish an entry to the Internet (a blog post)

Wasmo OS
--------

Support the preinstalled journal app.

Each app install gets its own database of entries.

Write published entries to the object store, and serve the published entries to the internet with a
public routes/ rule.

Support public and private routes, including redirects to sign-in.


Accounts
--------

Support sign-in and sign-out.
