Launcher Icons
==============

We're using web app (PWA) spec for our icons. We use the **maskable** icon described on [web.dev].

The icon should:

 - be square in aspect ratio
 - be at least 512px x 512px
 - look fine when center cropped to 80%, to a rounded square or circle shape

[web.dev]: https://web.dev/learn/pwa/web-app-manifest#icons
